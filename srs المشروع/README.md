# توجيه وثيقة المتطلبات البرمجية / Software Requirements Specification Pointer

- Full English SRS (>4,700 lines) is located at `Project_Documentation/EN/SRS.md` with endpoint-level code snippets and DTO validation matrices.
- النسخة العربية الكاملة متاحة ضمن `Project_Documentation/AR/SRS_AR.md` مع نفس التفاصيل.
- للاطلاع على خطة التنفيذ والتغييرات المصاحبة، يرجى مراجعة `Project_Documentation/EN/CIP.md` ونسختها العربية `Project_Documentation/AR/CIP_AR.md`.
- هذا المجلد يُحافظ على التوافق مع الاسم السابق ويعيد التوجيه إلى البنية الجديدة ثنائية اللغة.
